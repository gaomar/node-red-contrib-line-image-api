## 概要

LINEに投稿された画像を取得するNode-REDのノードです。

## インストール

```
npm i node-red-contrib-line-image-api
```

or

AdminタブからInstall

## 利用イメージ

### line-image
LINEから送られてくるMessageIDを取得して投稿された画像を返します。
プロパティ画面のMessageIDを指定すると対象IDの画像を取得することができます。
